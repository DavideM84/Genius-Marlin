
# Artillery Genius Marlin Firmware

## Firmware 3D-DavideM

Custom Marlin firmware for Artillery Genius printer.  
Marlin firmware v2.0.7.0  

## Build

Visual Studio Code  
PlatformIO  
Marlin Autobuild  

## Features

M600  
Junction Deviation  
Linear Advance  
G26 Bed test  
Manual Mesh Bed Level  
Print hours counter  
Adaptive step smoothing  
M701, M702  
S_Curve Acceleration  
Thermal Runaway  
Babystep
